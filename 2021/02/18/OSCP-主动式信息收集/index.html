    一：DNS 枚举

    1. DNS解析记录类型详解

    记录类型 描述
    A记录 将域名指向一个IPV4地址(例如：10.10.10.10)
    CNAME记录 如果将域名指向一个域名，实现与被指向域名相同的访问效果
    MX记录 建立电子邮箱服务，将指向邮件服务器地址
    NS记录 域名解析服务器记录，如果要将子域名指定某个域名服务器来解析
    TXT记录 可任意填写（可为空），通常用做SPF记录（反垃圾邮件）使用
    AAAA记录 将主机名（或域名）指向一个IPv6地址（例如：ff03:0:0:0:0:0:0:c1）
    SRV记录 记录了哪台计算机提供了哪个服务。格式为：服务的名字.协议的类型（例如：_example-server._tcp）
    显性URL 将域名指向一个http（s)协议地址，访问域名时，自动跳转至目标地址（例如：将www.flao.cn显性转发到www.laosec.cn后，访问www.falao.cn时，地址栏显示的地址为：www.laosec.net）
    隐形URL 与显性URL类似，但隐性转发会隐藏真实的目标地址（例如：将www.falao.cn隐性转发到www.laosec.cn后，访问www.falao.cn时，地址栏显示的地址仍然为：www.laosec.cn）

    2. Host命令

    把一个主机名解析到一个网际地址或把一个网际地址解析到一个主机名

    参数 描述
    -a 等价于使用”-v -t”
    -c Class 当搜索非网际数据时要指定查找的类
    -d 打开调试方式
    -n 等价于发布/usr/bin/hostnew命令.hostnew命令是host命令的5.2版本
    -r 禁用递归处理
    -t Type 指定要查询的记录类型
    -v 详细方式
    -w 永远等待 DNS服务器的一个回答
    -z 使用显示资源记录信息的新输出 
    host megacorpone.com   #默认查询A记录
host -t txt megacorpone.com  #指定查找TXT记录

    5. DNS区域传输

    区域传输基本上是在区域所在的相关DNS服务器之间进行的数据库复制文件从主DNS服务器复制到从服务器。区域文件包含所有DNS的列表为该区域配置的名称。区域传输应该只允许被授权的从属DNS但是许多管理员错误地配置了他们的DNS服务器，在这种情况下，任何人对于DNS服务器区域的副本，通常会收到一个。这相当于把公司网络布局交给黑客。所有的名字,服务器的地址和功能可能会被窥探.

    二：端口扫描

    端口扫描是检查远程机器上的TCP或UDP端口的过程，目的是检测目标上正在运行的服务以及可能存在的潜在攻击面还必须了解端口扫描的含义，以及特定端口扫描可能产生的影响。由于某些扫描可能产生的通信量及其侵入性，盲目运行端口扫描可能会对目标系统或客户端网络产生不利影响，例如服务器和网络链接过载或触发IDS。运行错误的扫描可能会导致客户停机

    六：SNMP枚举

    1. SNMP扫描

    kali#nmap -sU --open -p 161 10.11.1.1-254 -oG open-snmp.txt 
# --open 只显示开放的端口
kali@kali:~$ echo public > community
kali@kali:~$ echo private>> community
kali@kali:-$ echo manager >> community
kali@kali:~$ for ip in $(seq 1 2S4); do echo 10.ll.1.$ip; done > ips
kali@kali:~$ onesixtyone - c community -i ips

    我们可以使用snmpwalk之类的工具来探测和查询SNMP值，前提是我们至少知道SNMP只读字符串，在大多数情况下它是“public”

    • 枚举MIB树

      kali#snmpwalk -c public -v1 -t 10 10.11.1.14 
# -c 指定字符串 -v 指定SNMP版本号 -t 10 设置超时时间为10秒

    • 枚举windows用户

      kali#snmpwalk -c public -v1 10.11..1.14 1.3.6.1.4.77.1.2.25

    • 枚举正在运行的windows进程

      kali#snmpwalk -c public -v1 10.11.1.73 1.3.6.1.2.1.25.4.2.1.2

    • 枚举开放的TCP端口

      kali#snmpwalk -c public -v1 10.11.1.14 1.3.6.1.2.1.6.13.1.3

    • 枚举安装的软件

      kali#snmpwalk -c public -v1 10.11.1.50 1.3.6.1.2.1.25.6.3.1.2
